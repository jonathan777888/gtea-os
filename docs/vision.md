# Vision du projet GTEA OS

GTEA OS est un assistant IA personnel pour réussir la formation GTEA.

Le projet vise à aider l’étudiant à :
- mieux comprendre les cours ;
- transformer les consignes en actions concrètes ;
- réviser plus efficacement ;
- produire des réponses structurées ;
- suivre les devoirs et remises.

Le MVP doit d’abord permettre d’importer un document, de le résumer, de créer une checklist et de générer 10 questions de révision.cd gtea-os
